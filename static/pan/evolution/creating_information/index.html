But mutation can only destroy information!

This is a common misunderstanding. I think it's because when we normally see mutation at work, it's obviously detrimental, like when it leads to Cystic Fibrosis or Down Syndrome. The problem is that's like walking down a dark street and only hearing a single dog barking. You know that house contains a dog, but you have no information about the dog-having-ness of any other homes.

Most mutation is, as this intuition confirms, detrimental. Much of it (for example mutations in non-coding DNA) would have no effect, and some of it will be weakly beneficial. For mutations in the latter category, we have no way of knowing that they're present unless they actually help us out. (For example, by making us more resistant to a certain disease.)

Also, mutation is only one mechanism by which information is created. Sexual reproduction, for example, is another. Merging two sets of genes creates a functional organism with a completely unique code. Though this is a remix, it's still "new information".
